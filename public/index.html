Sobre o Projeto

Projeto aberto da comunidade Dev-ES com a finalidade de criar um serviço de rastreio geográfico de carros roubados.

O principal objetivo é identificar a placa, consultar a api para saber o status do mesmo, se estiver marcado como roubado vamos gravar uma foto e a localização do GPS para reportar a ocorrência ao dono do carro. Futuramente, podemos considerar a parceria com os serviços de segurança local.
